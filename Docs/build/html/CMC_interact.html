2. Interaction with Coinmarketcap API

class Bot.CMC_API

Базовые классы: object

This object represents a CoinMarketCap interaction.

get_CMC_id(symbols)

Method returns CMC’s ids for symbols of currencies

Параметры

symbols (str) – Comma-separated list of symbols of cryptocurrencies to check.

Результат

Comma-separated list of ids of cryptocurrencies to check.

Тип результата

str

get_CMC_rank(symbols)

Method returns CMC ranks for setted cryptocurrencies

Параметры

symbols (str) – Comma-separated list of symbols of cryptocurrencies to check.

Результат

A map where:

keys: Symbols of token.

values: Maps where:

link: A CMC token link.

name: A full name of token.

rank: Rank.

Тип результата

map

get_market_pairs(symbols)

Method returns CMC number of market pairs for setted cryptocurrencies

Параметры

symbols (str) – Comma-separated list of symbols of cryptocurrencies to check.

Результат

A map where:

keys: Symbols of token.

values: Maps where:

link: A CMC token link.

name: A full name of token.

pairs: Number of pairs.

Тип результата

map

get_price_change(symbols, period='1h', convert='USD')

Method returns price changings for a setted period

Параметры

  • symbols (str) – Comma-separated list of symbols of cryptocurrencies to check.

  • period (str, optional) – Comma-separated list of periods of periods to check. Can be „1h“, „24h“, „7d“, „30d“. Defaults to „1h“.

  • convert (str, optional) – Comma-separated list of symbols of crypto(currencies) for converting. Defaults to „USD“.

Результат

A map where:

keys: Symbols of token.

values: Maps where:

link: A CMC token link.

name: A full name of token.

price: A map where:

keys: Symbols of crypto(currency).

values: Maps where:

price: Current price.

changes: A map where:

keys: Period.

values: Changes.

Тип результата

map
